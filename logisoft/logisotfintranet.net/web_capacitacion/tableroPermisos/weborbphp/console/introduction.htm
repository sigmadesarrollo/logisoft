WebORB for PHP
 
 build 3.6.0 - June 2009		  

 

WebORB for PHP is an integration server enabling connectivity between Flex, Flash and Silverlight clients and PHP applications. Using WebORB, supported rich clients clients can invoke remote PHP functions and receive invocation results. Additionally, WebORB for PHP provides an implementation of a data management and message exchange systems. The product includes a powerful code generator for creating both client-side (ActionScript) and server-side (PHP) code to enable all database operations (data retrieval, creation, deletion and modification).

The messaging subsystem enables client applications to publish messages to destinations and WebORB delivers them to all subscribers.

WebORB for PHP must be deployed in a web server and have access to the user PHP classes (via special /Services folder). It functions as a gateway or Object Request Broker (ORB) between rich clients and the backend systems. WebORB implements the AMF0 and AMF3 protocols and thus can process requests from both Flex and Flash clients.
