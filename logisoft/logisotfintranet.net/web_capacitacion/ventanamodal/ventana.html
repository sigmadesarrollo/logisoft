* Este ejemplo abrirá una ventana-modal con tamaño variable. La variación del tamaño dependerá de las dimensiones de la ventana del navegador, es decir, cuanto más grande sea la ventana del navegador web, más grande será nuestra ventana-modal y, lo mismo cuando es pequeña: cuanto más pequeña sea la ventana del navegador, más pequeña será nuestra ventana-modal. La variación de tamaño sólo se hará al abrir la ventana-modal, de tal manera que si redimensionamos la ventana del navegador web una vez abierta la ventana-modal, esta última no variará de tamaño, pero al volverla a abrir sí variará adaptándose a las nuevas medidas.

La función que abre esta ventana es abrirVentanaVariable(pagina, nombre, titulo):
El parámetro pagina representa el nombre del archivo HTML (con diversas extensiones .HTML, .HTM, .JSP, .PHP, .ASP...) o bien la ruta absoluta de una url de internet (por ejemplo: "http://www.google.es/").

El parámetro nombre representa el atributo name del iframe (por ejemplo: "ventana-modal")que insertaremos en la ventana (página o dirección web que hemos indicado en el parámetro anterior) de forma que podamos identificar el objeto, para, por ejemplo, acceder a éste desde una función JavaScript. Esto se traducirá en el siguiente código HTML: <iframe name="ventana-modal">.

El parámetro titulo representa el título de la ventana (por ejemplo: "Google").



** Este ejemplo abrirá una ventana-modal con tamaño fijo. El tamaño será el que decidamos en la llamada a la función JavaScript, es decir, es un tamaño fijo que no dependerá de la ventana del navegador, pero que sí podremos variar en cada llamada a la ventana-modal.


La función que abre esta ventana se encuentra en el archivo abrir-ventana-fija.js y es abrirVentanaFija(pagina, ancho, alto, nombre, titulo):
El parámetro pagina cumple la misma función que en el caso anterior.

El parámetro ancho representa el ancho que queremos que tome la ventana-modal. Deberá ser un valor numérico (sin comillas simples o dobles). Hemos de procurar que el ancho no sea superior al documento HTML que tenemos para que el diseño no descuadre la ventana-modal.

El parámetro alto representa el alto que queremos que tome la ventana-modal. Deberá ser un valor numérico (sin comillas simples o dobles). Lo mismo ocurre que con el ancho: debemos procurar que no sea superior al alto del documento HTML para que el diseño no se descuadre.

El parámetro nombre cumple la misma función que en el caso anterior.

El parámetro titulo cumple la misma función que en el caso anterior.



Hay que tener presente que estos dos ejemplos son sólo eso: ejemplos. La funcionalidad dependerá de las necesidades de cada programador. Se podría utilizar ventana-modal como script para hacer que la página principal de nuestra web quede en el centro o bien para mostrar un cargador (loading...) en el centro de la pantalla y evitar que el usuario pueda acceder a otro contenido de la web mientras está cargando.

El código es libre y se puede utilizar y modificar tanto como uno quiera. Se recomienda no modificar el esqueleto (ventana-modal-1.1.1.js) ya que en futuras revisiones, mejoras, etc será más difícil el proceso de actualización. Las imágenes y estilos de la ventana-modal no son libres y pertenecen a sus determinados autores, por lo que no debe utilizarse para proyectos propios o comerciales, pero sí puede servir de guión para crear imágenes y estilos propios.
