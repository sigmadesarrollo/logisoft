Example name: Flex Chat using WebORB Messaging
Description: The example demonstrates the following:
  • Using the WeborbProducer/WeborbConsumer API for publish/subscribe messaging
  • Ability to broadcast to a destination
  • Ability to retrieve messages from a destination
Client-side code: Browse  Download
Server-side code: no custom code required
Run example: Open additional chat client
Key points:
  • Client project must import weborb.swc to obtain access to the WeborbConsumer and WeborbProducer API.
  • Each client includes one producer and one consumer object. Both objects connect to the same destination.
