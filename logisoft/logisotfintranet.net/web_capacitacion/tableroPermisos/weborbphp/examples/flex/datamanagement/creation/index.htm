Example name: WebORB Data Management for Flex (WDMF) - Data Creation
Description: The example demonstrates the following:
  • How to create and save new active record in the database
  • How to find and monitor all records from the database
  • The example uses the following WDMF APIs:
    • ActiveRecord.save()
    • DataMapper.create( activeRecordInstance )
    • DataMapper.findAll()
Client-side code: Browse (use Back button to go back)  Download
Server-side code: No custom server-side code is required for this example. All the supporting code is generated by the WDMF code generator.
Things to try:
  • Open multiple instances of the client application in several windows. Create a new product, notice the grid is updated on all clients.
Run example:
Key points:
  • The example uses WDM API for instering new record to table
  • The save() operation checks if the record already exists in the database. If the record is found, save() update the record, otherwise it creates a new one. An alternative approach to save a new product is to use ActiveRecords.Product.create( productInstance ), the method performs the operation faster as it does not check if the record already exists.
